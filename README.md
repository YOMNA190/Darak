# Darak: Luxury Interior Design & Finishing Platform

Darak is a premium web application dedicated to luxury interior design and finishing services. Built with a modern tech stack, it provides a seamless and sophisticated experience for clients seeking high-end design solutions.

## 🚀 Key Features

*   **User Authentication:** Secure login and registration system for clients.
*   **Dynamic Content Management:** Efficient handling of project portfolios and service listings.
*   **Interactive UI/UX:** A visually stunning and responsive interface built with Framer Motion.
*   **Data Visualization:** Integrated charts for project progress and analytics using Chart.js.

## 🛠️ Tech Stack

*   **Frontend:** Next.js, React, TypeScript, Tailwind CSS, Framer Motion
*   **Backend & Database:** Supabase, PostgreSQL
*   **Authentication:** JWT, bcryptjs
*   **Other Tools:** Lucide React, React Hook Form, Chart.js

## 📦 Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/YOMNA190/Darak.git
    ```
2.  Install dependencies:
    ```bash
    cd Darak
    npm install
    ```
3.  Set up environment variables in a `.env.local` file.
4.  Run the development server:
    ```bash
    npm run dev
    ```
